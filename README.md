Para inicializar a aplicação, é necessário criar um banco de dados MySQL chamado "dbsysbanca" utilizando o seguinte comando:

CREATE DATABASE dbsysbanca;

Em seguida, acesse o diretório \teste\src\main\resources e abra o arquivo application.properties. No arquivo, configure o usuário e a senha conforme os dados registrados no MySQL. Após essas configurações, a aplicação funcionará normalmente.
